# Randomized_Seating_Plan---Assignment

An Assignment for an Object Oriented Programming class based on C++

This contains code for the OOP approach as well as the Procedural approach.
For both approaches, the information of passengers boarding and alighting the bus, on a route of five stops, was read from a file and the program would 
randomly assign seats to each passenger while adhering to constraints. These constraints were: Children are not allowed to be seated next to doors, 
male and female passengers must be separated and seated into the male section and women’s and family’s section, respectively.
